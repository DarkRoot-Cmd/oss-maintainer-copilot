# Contributing

Thanks for helping improve OSS Maintainer Copilot.

## Development setup

```bash
python -m venv .venv
. .venv/bin/activate
python -m pip install -e .
python -m unittest discover -s tests
```

## Pull request expectations

- Keep changes focused.
- Add tests for behavior changes.
- Update README or docs for user-facing changes.
- Be clear about security implications.

## Maintainer workflow

New issues should receive an initial label and priority. Pull requests should receive human review before merge, even when AI-generated summaries are used.
